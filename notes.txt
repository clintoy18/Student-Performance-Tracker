command to update dotnet ef:
dotnet tool update --global dotnet-ef

run migrations:
1. cd backend/ASI.Basecode.Data
2. dotnet ef migrations add <MigrationName>
3. dotnet ef database update

IF MIGRATIONS DONT WORK!
=It might be that dotnet-ef version does not match the project's: 9.0.6.
-To check version, do:
1. dotnet-ef --version
-To match version, do:
1. dotnet tool uninstall --global dotnet-ef
2. dotnet tool install --global dotnet-ef --version 9.0.6
=It could also be because you're not migrating in the right directory
-You must be in: backend/ASI.Basecode.WebApp
=If nothing else works, try to migrate in Visual Studio Nuget console
